============

Katalog programy zawiera jeden program w C++:
--------------------------------------------------------------------

I.  Program wypisujacy wszystkie funkcjonalności zadane w ćwiczeniu. 
    Program zawiera implementacje dla metod dla int-ów;
    Program sklada sie z jednego pliku .cpp:
    1) main.cpp  - program glowny zawierający implementację wszystkich
			funkcji: True/False
				 Mapa(wierzcholek, dlugosc sciezki),
				 Mapa(Wierzcholek, poprzednik)
		   program nalezy uruchomic w folderze, w ktorym 
     		   znajduje sie plik .csv lub umieścić plik z 
		   danymi w folderze "cmake-build-debug" i 
		   uruchomic program w domyślnym folderze.
				
    2) bellman.csv  - plik z danymi.
    2) bellman2.csv  - plik z danymi.
