 Zawartosc:
============

Katalog programy zawiera jeden program w C++:
--------------------------------------------------------------------

I.  Program wypisujacy wszystkie funkcjonalności zadane w ćwiczeniu. 
    Wypisuję także ilość operacji dominujących w w przypadku pesymistycznym.
    Program sklada sie z dwóch plików .cpp i jednego pliku .hpp: 
    1) main.cpp  - program glowny.
    2) AVLTree.cpp  - plik źródłowy z implementacją funkcji: insert, isEmpty, contains,
		      remove, printTreeInOrder,printTreePreOrder,printTreePostOrder, 
		      findMin, findMax. 
    3) AVLTree.hpp  - plik szablonowy z funkcjami.
